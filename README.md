# WhiskerWatch - Backend API

A Spring Boot REST API backend for a pet sitting booking platform. Connects pet owners with verified pet sitters through a secure, scalable API.

## Tech Stack

- **Spring Boot** - Java framework for enterprise applications
- **Java** - Core programming language
- **PostgreSQL** - Relational database (schema included)
- **REST API** - RESTful service architecture
- **Maven/Gradle** - Build and dependency management

## Features

- RESTful API endpoints for pet sitter bookings
- User authentication and authorization
- PostgreSQL database with normalized schema
- Secure data management for users, pets, and bookings
- Verified pet sitter profiles

## Database Schema

The `schema.sql` file contains a complete PostgreSQL database structure including:
- User management tables
- Pet profiles and preferences
- Booking and scheduling system
- Sitter verification and ratings
